A Drat repo

This is a drat repository. You can use it like this:

# once only
install.packages("drat")
drat::addRepo("jefferis")

# now you can install packages from this drat repo as well as CRAN
# NB it is recommended to use type='both' so that both source code
# and binary packages can be used.
install.packages("ANTsR", type="both")
You can see a package listing here
